# Workshop: creating a flexible navigation bar

**Theoretical Recap** Flexbox is a one-dimensional layout model that distributes space along a single axis — either row or column. The flex container is the parent element declared with `display: flex`, and its direct children automatically become flex items. The key properties to master for a navigation bar are: `flex-direction` (defines the main axis), `justify-content` (distributes items along the main axis), `align-items` (aligns items on the cross axis), `flex-wrap` (allows items to wrap to the next line), and `gap` (adds spacing between items without relying on margins). A common mistake beginners make is applying flex properties to the wrong element — remember that `justify-content` and `align-items` are set on the container, not the items. Another frequent error is forgetting to handle the mobile layout: a horizontal nav becomes a vertical stack on small screens using `flex-direction: column` inside a media query. Mental model: think of your nav as a flexible shelf — you control how items are placed, spaced, and whether they overflow or wrap. Mini example: `nav { display: flex; justify-content: space-between; align-items: center; flex-wrap: wrap; gap: 1rem; }`. This single block gives you a robust starting structure. --- **Workshop Brief** In this workshop, you will apply your Flexbox and responsive design skills to build a fully functional and adaptive navigation bar from scratch.
