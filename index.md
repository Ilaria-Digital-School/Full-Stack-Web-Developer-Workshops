Become a Full Stack Web Developer and master the development to create, manage and deploy complete applications (HTML, CSS, introduction to algorithms, JavaScript, NodeJS as the main backend, Python for scripting/automation, SQL, NoSQL, automation, SQL, NoSQL, automation, SQL, SQL, NoSQL, Git, SQL, Git, Git, unit tests, unit tests, secure your web application, CI/CD). Skills: Design a multi-layered and secure software architecture (presentation, business, persistence); Develop the front-end and back-end components of a web application; Implement secure REST APIs interconnected to a relational or NoSQL database; Automate deployments via CI/CD pipelines in a DevOps logic; Write technical documentation and prepare for production; Collaborate in Agile project mode (user stories, backlog, sprints, standalone); -ups);
